# Deep Learning Based Road Pothole Detection System

A Convolutional Neural Network (CNN) based image classification system that detects whether a road image contains potholes.

## Features

- Binary Image Classification
- CNN Architecture
- TensorFlow/Keras
- Confusion Matrix
- Accuracy & Loss Graphs
- Prediction on Custom Images

## Dataset

Dataset consists of two classes:
- Normal Road
- Pothole Road

Images are resized to 128x128 pixels.
